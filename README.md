# 多维表格附件视频预览插件

这个插件用于在飞书多维表格右侧栏中预览附件字段里的视频文件，支持通用模式（任意多维表格）。

## 功能说明

- 自动监听当前选中单元格。
- 当选中附件单元格时，自动解析该格中的视频附件并播放。
- 若自动识别失败，可切换到手动模式并指定附件字段。
- 一个单元格有多个视频时，默认播放第一个，并可在列表中切换。
- 非视频附件会自动过滤，仅保留可播放视频。

## 本地启动

```bash
npm install
npm run dev
```

## 打包发布

```bash
npm run build
```

构建产物位于 `dist/`，可用于上传到飞书插件托管环境。

## 接入说明（飞书多维表格）

1. 在飞书开放平台创建多维表格扩展（侧边栏类型）。
2. 将开发地址配置为本地 `vite` 地址进行调试，或上传 `dist/` 产物进行发布。
3. 在多维表格中安装该扩展后，点击任意记录的附件单元格即可在右侧预览视频。

## 已知限制

- 需要插件对当前表有读取附件权限。
- 依赖多维表格返回可用附件 URL，权限不足时会显示错误信息。
- 某些非常规视频格式可能受浏览器编解码能力限制。
